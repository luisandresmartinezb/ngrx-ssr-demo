# ngrx-ssr-demo

Proyecto demo desarrollado con Angular para explorar el uso de **NgRx** y **Angular SSR**.

## Objetivo
Analizar cómo NgRx ayuda en la gestión del estado y cómo Angular SSR mejora el rendimiento inicial y el SEO.

## Tecnologías
- Angular
- NgRx
- Angular SSR
- TypeScript

## Ejecución
```bash
npm install
npm run dev:ssr
